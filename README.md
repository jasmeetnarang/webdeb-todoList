webdev-todoList
===============

###Usage

1. Clone Repo:
 
       git clone https://github.com/jasmeetnarang/webdev-todoList.git

2. Browse to clone directory

        cd webdev-todoList
    
3. Install NPM packages
        
        npm i
4. Start Node Server
 
        node sever.js
       
5. Browse to http://localhost:8080 or see live at https://pacific-chamber-32466.herokuapp.com/
